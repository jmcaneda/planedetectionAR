# planedetectionAR
Detección de planos, movimiento y comportamiento de gizmo con AR en Unity
M-ART Soho se basa en parte, en esta experiencia previa y en otros desarrollos para la creación del PortalAR.
Soy un entusiasta de las TIC en general, su aplicación en Realidad Aumentada y las posibilidades que nos brinda como medio de interacción, con realidades ficticias.
Podeis descargar la experiencia M-ART Soho desde https://jmcaneda.itch.io/m-art-soho
Teneis las herramientas de desarrollo en https://alhilodigital.blogspot.com/
Solo para Android, por ahora.
